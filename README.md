# Cassette Music Player
[![License: GPL v3](https://img.shields.io/badge/License-GPL%20v3-blue.svg)](https://github.com/TheTerminatorOfProgramming/CassetteMusicPlayer/blob/master/LICENSE.txt)

<a href='https://play.google.com/store/apps/details?id=com.ttop.cassettemusicplayer'><img alt='Get it on Google Play' src='https://play.google.com/intl/en_us/badges/images/generic/en_badge_web_generic.png' height="90"/></a>

**A material designed local music player for Android.**

**Forked from [Vinyl](https://github.com/AdrienPoupa/VinylMusicPlayer)**

Additional features:
* Redesigned Widgets
* System Default Theme (Switch Theme Based On your Device Theme)
* Removed Navigation Drawer
* Fast-Forward and Rewind Functions
* [...and more!](https://github.com/TheTerminatorOfProgramming/CassetteMusicPlayer/blob/master/CHANGELOG.md)
